🚀 Features Implemented

  • ✅ Mastodon API v1 (accounts, statuses, timelines)
  • ✅ AT Protocol (DID documents, session management)
  • ✅ ActivityPub Federation (WebFinger, actors, inbox/outbox)
  • ✅ WebSocket Streaming (real-time updates)
  • ✅ Full-text Search (posts, users, hashtags)
  • ✅ OAuth2 Authentication
  • ✅ Media Upload & Processing
  • ✅ Rate Limiting & Caching
  • ✅ Background Jobs (email, federation)
  • ✅ Moderation (blocking, muting, reports)
  • ✅ Polls & Voting System
  • ✅ Bookmarks & Lists
  • ✅ Featured Posts & Emoji Reactions
  • ✅ Admin Dashboard & Statistics

🛠️ Tech Stack

  • Backend: Zig (high-performance, memory-safe)
  • Database: SQLite (embedded, fast)
  • HTTP Server: std.http (built-in Zig)
  • WebSockets: Custom implementation
  • Frontend: HTMX (minimal, fast)
